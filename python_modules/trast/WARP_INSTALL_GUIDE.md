# 🛠️ Исправленный скрипт установки WARP

## ✅ Что исправлено:

1. **Правильные команды**: Используется `warp-cli mode proxy` вместо `warp-cli set-mode proxy`
2. **Улучшенная обработка ошибок**: Скрипт не падает при ошибках регистрации
3. **Безопасность**: Четкие предупреждения о безопасности
4. **Тестирование**: Улучшенное тестирование подключения
5. **Информативность**: Подробные сообщения о статусе

## 🚀 Как использовать:

### Установка WARP (безопасная):
```bash
sudo ./install_warp.sh
```

### Быстрое исправление (если что-то пошло не так):
```bash
sudo ./quick_fix_warp.sh
```

### Экстренное исправление:
```bash
sudo ./emergency_fix.sh
```

## 🔒 Гарантии безопасности:

- ✅ WARP устанавливается в proxy-only режиме
- ✅ SSH и сеть НЕ блокируются
- ✅ Можно безопасно отключить в любой момент
- ✅ Работает только как прокси на порту 40000

## 📋 Команды управления:

```bash
warp-cli mode proxy      # Установить безопасный режим
warp-cli connect         # Подключиться
warp-cli disconnect     # Отключиться
warp-cli status         # Проверить статус
warp-cli settings list  # Показать настройки
```

## 🎯 Результат:

После установки парсер будет использовать:
1. **WARP** (приоритет) - если доступен
2. **Tor + прокси** - если WARP недоступен  
3. **Прокси** - если Tor недоступен

## ⚠️ Важно:

- Никогда не используйте `warp-cli mode warp` на сервере!
- Всегда используйте `warp-cli mode proxy`
- При проблемах запустите `quick_fix_warp.sh`
