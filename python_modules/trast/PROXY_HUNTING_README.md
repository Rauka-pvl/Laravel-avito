# 🎯 Мощный поиск прокси для Trast Parser

Этот набор скриптов предназначен для поиска и тестирования рабочих прокси с локальной машины.

## 📁 Файлы

- **`proxy_hunter.py`** - Основной скрипт для поиска прокси
- **`test_proxies_fast.py`** - Быстрое тестирование прокси из файла
- **`upload_proxies.py`** - Загрузка найденных прокси на сервер

## 🚀 Использование

### 1. Поиск прокси (основной скрипт)

```bash
cd python_modules/trast
python3 proxy_hunter.py
```

**Что делает:**
- Получает прокси из 13 источников
- Тестирует их агрессивно (батчами по 200)
- Сохраняет результаты в JSON и TXT файлы
- Ищет до 200 рабочих прокси

**Выходные файлы:**
- `working_proxies_YYYYMMDD_HHMMSS.json` - детальные результаты
- `proxy_test_results_YYYYMMDD_HHMMSS.json` - полная статистика
- `proxies_for_server_YYYYMMDD_HHMMSS.txt` - простой список для сервера

### 2. Быстрое тестирование

```bash
python3 test_proxies_fast.py proxies_for_server_20241017_120000.txt 100
```

**Параметры:**
- `proxies_for_server_20241017_120000.txt` - файл с прокси
- `100` - количество прокси для тестирования (опционально)

### 3. Загрузка на сервер

```bash
python3 upload_proxies.py working_proxies_20241017_120000.json root@31.172.69.102
```

**Параметры:**
- `working_proxies_20241017_120000.json` - файл с прокси
- `root@31.172.69.102` - данные сервера

## ⚙️ Настройки

В `proxy_hunter.py` можно изменить:

```python
# Количество рабочих прокси для поиска
await hunter.hunt_proxies(max_working=200)

# Размер батча для тестирования
batch_size = 200

# Таймауты для тестирования
timeout=httpx.Timeout(3.0, connect=2.0)
```

## 🔧 Рекомендации

1. **Запускайте на быстром интернете** - тестирование требует много трафика
2. **Используйте VPN** - некоторые прокси могут быть заблокированы
3. **Запускайте ночью** - процесс может занять несколько часов
4. **Проверяйте результаты** - не все найденные прокси могут работать на сервере

## 📈 Ожидаемые результаты

- **Найдено прокси**: 40,000+ из источников
- **Протестировано**: 2,000-5,000 прокси
- **Рабочих найдено**: 50-200 прокси
- **Время работы**: 1-3 часа
- **Процент успеха**: 2-5%
