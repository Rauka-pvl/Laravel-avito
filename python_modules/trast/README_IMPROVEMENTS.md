# Trast Parser - Улучшенная версия

## 🚀 Основные улучшения

### ✅ Исправленные проблемы:
- **Синтаксические ошибки** - исправлены все ошибки отступов и структуры кода
- **Проблема с ChromeDriver** - добавлено автоматическое управление версиями
- **Блокировка по IP** - убран fallback на прямое подключение
- **Улучшенная обработка ошибок** - добавлены множественные попытки и детальная диагностика

### 🔧 Новые возможности:
- **Поддержка Tor** - автоматическое переключение на Tor при недоступности прокси
- **Улучшенная ротация прокси** - увеличено количество проверяемых прокси до 500
- **Детальная диагностика** - подробные логи для отладки проблем
- **Автоматическое восстановление** - умная система бэкапов и восстановления

## 📋 Требования

### Python зависимости:
```bash
pip install -r requirements.txt
```

### Tor (опционально, но рекомендуется):
```bash
# Установка Tor
sudo apt install tor
sudo systemctl start tor
sudo systemctl enable tor

# Или используйте готовый скрипт:
chmod +x install_tor_simple.sh
./install_tor_simple.sh
```

## 🎯 Как работает

1. **Проверка прокси** - скрипт проверяет до 500 прокси из списка
2. **Создание драйвера** - использует undetected-chromedriver с автоматическим управлением версиями
3. **Fallback на Tor** - если прокси не работают, автоматически переключается на Tor
4. **Установка сессии** - множественные попытки установки легитимной сессии
5. **Парсинг данных** - умная система задержек и человеческое поведение

## 🔍 Диагностика проблем

### Если скрипт не запускается:
```bash
# Проверьте синтаксис
python -m py_compile trast/main.py

# Проверьте зависимости
pip list | grep -E "(undetected-chromedriver|selenium|requests)"
```

### Если не работает парсинг:
1. **Проверьте Tor**: `sudo systemctl status tor`
2. **Обновите прокси**: добавьте новые рабочие прокси в файлы
3. **Проверьте логи**: смотрите детальные логи в консоли

### Если блокировка по IP:
- Скрипт автоматически использует только прокси/Tor
- Прямое подключение полностью отключено
- При отсутствии рабочих прокси пробует Tor
